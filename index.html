Sorted Array

when the length of arr is odd, like [1, 2, 3] we know mid = 2without any problem. But what if the length of arr is even? like [1, 2], below is the logic :

  1. since mid = (left + right ) / 2 mid firstly, will equal to the very left one 1
  2. If target is 2, since left still less than right, we will still jump into while loop
  3. Then, left still less than target, and left will be assigned to mid + 1 = 0 + 1 = 1 = right, the scope turn into [2]
  4. NOTE: left <= right also means, scope >= 0. Even right - left = 0 mid = right / left.

while(left <= right) 的终止条件是 left == right + 1，写成区间的形式就是 [right + 1, right]，或者带个具体的数字进去 [3, 2]，可见这时候区间为空，因为没有数字既大于等于 3 又小于等于 2 的吧。所以这时候 while 循环终止是正确的，直接返回 -1 即可。

while(left < right) 的终止条件是 left == right，写成区间的形式就是 [right, right]，或者带个具体的数字进去 [2, 2]，这时候区间非空，还有一个数 2，但此时 while 循环终止了。也就是说这区间 [2, 2] 被漏掉了，索引 2 没有被搜索，如果这时候直接返回 -1 就是错误的。

    // ...
}
